reason

Taiwan has a lot of interns every year, but there are no such a platform collect all kinds of review of intern. I build this platform with a team, follow with their git flow, use Google Sprint method to find out how user react with the platform and where should fix by their feedback.

skill

Use python parser fetching raw data from Google spread sheet. Firebase as backend for provide api data. using React-Redux as frontend develop flow. an platform with frontend, backend, UI all done by myself.
